Release Notes for V1.0

Launch Setup.exe to install the software.
Follow the on screen dialog boxes for installation procedure.

---------------------------------------------------
What's new in Release 1.0?
---------------------------------------------------
This is the first release of the Motor Control Demo.

It supports following configurations.
 - BLDC Motor Open Loop Control on PICDEM MC
 - BLDC Motor Close Loop Control on PICDEM MC
 - BLDC Motor Open Loop Control on PICDEM MC with 3Ph Current Measurement
 - BLDC Motor Close Loop Control on PICDEM MC with 3Ph Current Measurement
 - BLDC Motor Open Loop Control on PICDEM MC LV
---------------------------------------------------
What's new in Release 1.0.1?
---------------------------------------------------
 Fixed random Abort on Stop issue.
 Temperature calibration support for newer PIC18 Firmware (V1.1 & higher)

---------------------------------------------------
What's new in Release 1.0.2?
---------------------------------------------------
